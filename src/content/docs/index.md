## Start building

- [Quickstart](/getting-started/quickstart/) — Install icp-cli and deploy your first canister

## What makes ICP unique

- [Chain-Key Cryptography](/concepts/chain-key-cryptography/) — Sign Bitcoin and Ethereum transactions natively
- [VetKeys](/concepts/vetkeys/) — On-chain encryption and key management
- [HTTPS Outcalls](/concepts/https-outcalls/) — Call any Web2 API from a canister
- [Reverse Gas Model](/concepts/reverse-gas-model/) — Users never pay gas fees
